App Reskin Documentation

This guide provides step-by-step instructions to reskin the "Crypto Pulse" Flutter application. The app is built using Flutter and supports both Android and iOS platforms.

1. Understanding the Current App Structure

The app is a cryptocurrency portfolio manager built with Flutter. It uses:

  • Hive for local storage.
  • Pie Chart package for graphical representation.
  • Material Design for UI components.

Key files and functions:

  • Main App (main()): Initializes Hive and sets up the app.
  • PortfolioScreen: The main screen, displaying the portfolio and pie chart.
  • Crypto Model: Represents individual cryptocurrency entries.

2. Planning the Reskin

Reskinning involves modifying:

  • UI Design: Colors, fonts, themes.
  • Branding: App name, icon, and splash screen.
  • Assets: Images, fonts, and other visual elements.

3. Steps to Reskin the App

Step 1: Update the App Name

1. Open android/app/src/main/AndroidManifest.xml for Android.

2. Modify the <application> tag:

<application
    android:label="New App Name"
    ... >

3. For iOS, edit ios/Runner/Info.plist:

<key>CFBundleName</key>
    <string>New App Name</string>

Step 2: Change the App Icon

1. Replace the default icon using a tool like App Icon Generator.

2. Place the generated icons in the respective platform directories:

  • Android: android/app/src/main/res/mipmap-*
  • iOS: ios/Runner/Assets.xcassets/AppIcon.appiconset

Step 3: Modify the Splash Screen

1. For Android, update the splash screen in android/app/src/main/res/drawable/launch_background.xml.

2. For iOS, modify the LaunchScreen.storyboard file in ios/Runner/Base.lproj.

4. Testing the Reskinned App

1. Run the app on Android and iOS emulators and real devices to ensure:

  • The UI matches the new design.
  • There are no broken assets or layout issues.

2. Check the functionality of:

  • Adding, editing, and deleting cryptocurrencies.
  • Pie chart display and interactions.
  • Multi-currency support.

5. Build and Release

1. Build the app for Android and iOS:

  • Android: flutter build apk
  • iOS: flutter build ios

2. Test the builds on actual devices.

3. Submit the app to the respective app stores:

  • Google Play Store for Android.
  • Apple App Store for iOS.

Conclusion

Following these steps ensures a complete reskin of the "Crypto Pulse" app for both Android and iOS platforms. Adjustments can be made to align with branding and design requirements. Happy coding!
